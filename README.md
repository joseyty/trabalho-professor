# trabalho-professor
Bom dia. Neste projeto desenvolvi um portfólio de Analista Júnior em Mineração de Dados utilizando Python e o framework Streamlit. O objetivo principal foi simular uma atividade real da área de análise de dados, desde a pesquisa de vagas até a criação de uma aplicação capaz de apresentar informações e gerar insights a partir de uma base de dados.
